Cloud Sync Setup

Share the same Gist ID on both your devices to see each other's foods in real-time!

How to Setup Cloud Sync:

  1. One of you goes to gist.github.com
  2. Create a secret gist with any filename and content
  3. Copy the Gist ID from the URL (the long hash after your username)
  4. Both of you enter the same Gist ID in the field above
  5. Now you'll see each other's foods automatically!

Example URL: https://gist.github.com/username/a1b2c3d4e5f67890abcd1234ef567890
